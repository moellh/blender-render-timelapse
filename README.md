# Render-Timelapse

Blender add-on which generates timelapse video of rendered scene from recorded edits in Blender.

- See [BLOG.md](./BLOG.md) for changes made during development.
- See [TODO.md](./TODO.md) for open (& closed) tasks that need to be implemented.

## Local Installation

1. Go to `Edit > Preferences > Get Extensions > Add Extensions Repository > Add Local Repositor`.
2. Set `Name`, e.g. to "Render Timelapse".
3. Activate `Custom Directory` and set to directory of the repo.
4. Press `Create`.
5. Go to `Add-ons`, find Render Timelapse, and activate it.
